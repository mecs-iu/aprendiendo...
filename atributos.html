Atributos

PRINCIPAL

Algunos atributos globales que están disponibles para la mayoría de las etiquetas en HTML


class: Este atributo se usa para dar estilos a través de css

Ejemplo:

se escribe en HTML--- <p class="logo">Logo</p> se encuentra en CSS--- .logo


id: Es un identificador único que se utiliza para seleccionar el elemento desde JS y para hacer navegación a través de anclas.

Ejemplo:

se escribe en HTML--- <p id="logo">Logo</p> se encuentra en CSS--- #logo


title: Es un atributo que ayuda a la accesibilidad mostrando una descripción del elemento al que pertenece. Cuando se usa aparece el mensaje en un tooltip.

Ejemplo:

se escribe en HTML--- <p title="Este es un ejemplo del atributo title">Título con tooltip</p>


data: Es un atributo que nos permite guardar algún valor en la etiqueta HTML.


Atributos de los enlaces

target: Define donde se abrirá el recurso solicitado. Por norma general al usar rutas absolutas pondremos como valor "_blank"

Ejemplo:

<a href="https://google.com" target="_blank">ir a google < a >


download: Atributo booleano, sirve para descargar el recurso solicitado.

IMPORTANTE: el recurso debe estar en nuestro mismo servidor

Ejemplo:

<a href="/paginas/contacto.html"download>ir a google < a >

<a href="/assets/images/gato.jpg"download> < a >

NOTA: Si el archivo fuera externo no lo descargaría y si fuera imagen en vez de descargarlo lo abriría
