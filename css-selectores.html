Selectores

PRINCIPAL

Selectores elementales

Selector universal -> * { }

Selector de tipo o etiqueta -> ej: h1 {}


Selectores de atributo

class: Este atributo se usa para dar estilos a través de css

Ejemplo:

se escribe en HTML--- <p class="logo">Logo</p> se encuentra en CSS--- .logo


id: Es un identificador único que se utiliza para seleccionar el elemento desde JS y para hacer navegación a través de anclas.

Ejemplo:

se escribe en HTML--- <p id="logo">Logo</p> se encuentra en CSS--- #logo


[]: Este es otro modo de dar estilo a través de corchetes, estos se crean con Alt 91 y Alt 93, y para modificar unos enlaces sería en css:

[href]{background-color:red;}

Si hay cuatro enlaces podemos darle estilo a uno solo de la siguiente manera: con atributo mas valor
[href="https://google.com"] {background-color: red;}
