Etiquetas importantes de bloque

PRINCIPAL

address: Se utiliza para aportar información de contacto para el article más cercano o para todo el body.


Ejemplo:
Tony Stark
Malibú 10880
90265
California

blockquote: Se utiliza para marcar las citas a otros autores o documentos


        Ejemplo:
        

            

                Lorem ipsum dolor sit amet consectetur adipisicing elit. Vero veniam ullam quisquam dignissimos, deserunt omnis debitis qui provident iste molestias eveniet excepturi suscipit explicabo. Pariatur dicta mollitia quas in harum.

pre: Se utiliza para tener código preformateado que necesita ser representado igual que se escribió


Ejemplo: 
     Hola 
     ¿Que tal?

si movemos con el tabulador una palabra también se mueve en el texto.

                Hola
    ¿Que tal?

div: Se usa como división del documento, semánticamente no significa nada, es un contenedor genérico que normalmente se usa para dar estilos a través de css o para usar algo denominado "delegación de eventos" en javascript

Ejemplo:

Nombre

email

Descrpción

Nombre

email

Descrpción


hr: horizontal rule, se utiliza para decirle al navegador que vamos a cambiar de tema


span: Es un contenedor de línea, equivalente a div, se suele utilizar para encerrar palabras o pequeños textos y darles estilo a través de css o localizarlos desde javascript. Semánticamente no significa nada.

Ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Quisquam reprehenderit autem deserunt saepe officia sint eaque fugiat rerum est porro? Rerum earum magni harum corporis, nobis tempora labore sit voluptatum.


q: Es el equivalente a blockquote, significa quote, por eso el de bloque se llama block-quote. Sirve para poner citas pero en líneas.

Ejemplo: Yo soy Ironman

code: Sirve para encerrar código que queremos representar visualmente, suele ir unido con la etiqueta pre

Ejemplo: 
    
        span{
            color: red
        }
    
    
Para describir en algunos casos partes de códigos usando < y > sería con & # 60 ; y  & # 62 ; 

    
 &+Ctrl+space abre la lista de símbolos del teclado

Etiquetas de línea

em - enfasis


strong - mas enfasis


small - menos enfasis


br - salto de línea


wbr - salto de línea si hiciera falta


time - 24/10/2020 10:35


i - italic


b - bold


u - underline

sup - 42

sub - H 2O
