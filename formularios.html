Formularios

PRINCIPAL

La estructura básica de un formulario se compone de 4 elementos:

form: es la etiqueta que engloba nuestro formulario
label: sirve para escribir el nombre del campo a rellenar
input: sirve para crear un campo que permitirá al usuario interactuar
button: crea un botón que permitirá enviar el formulario

Elementos seleccionables

radio

Permite seleccionar una única opción de una lista de opciones relacionadas

Género

* La opción name ="gender" fuerza al código a seleccionar sólo uno.

También está el atributo booleano checked que, al colocarlo solo en una opción hace que quede seleccionado por defecto


checkbox

permite seleccionar varias opciones de una lista de opciones relacionadas

Lenguajes que conoces

type="checkbox" es para crear el casillero seleccionable

value="Languajes" es para decirle al navegador de que es el conjunto que estamos seleccionando

value="html" es para informar el nombre de lo que se seleccionó


select

Crea una lista de opciones donde podemos seleccionar una o varias opciones

Cada opción irá dentro de una etiqueta <option> </option>


De este modo podremos seleccionar solo una de las opciones que figuran, para seleccionar más de una ...


De este modo en vez de ser un desplegable pasa a ser una lista

Con el atributo booleano multiple podremos seleccionar una opción o si queremos seleccionar mas opciones debemos hacerlo con "Ctrl" presionado

Elemento select avanzado

Si tenemos muchas opciones podemos ordenarlas por categorías a través de la etiqueta <optgroup> con el atributo "label" para nombrar la categoría


datalist

Es muy similar a select pero incluye un filtro para que se pueda buscar de una forma mas sencilla

Con datalist queda el dato que da "value" y podemos dejar opcional el mensaje a modo de subtítulo ó información


fieldset

Se utiliza para agrupar elementos dentro de un formulario


legend

Representa una etiqueta para el contenido del fieldset

Atributos para formularios

placeholder

Da una pista de lo que el usuario debe introducir


readonly

hace que un campo sea de solo lectura

disabled

Desactiva el campo, no se podrá escribir en él

min - max

Establece el mínimo y máximo de un campo numérico

minnlenght - maxlenght

Establece el mínimo y máximo de caracteres de un campo de texto

selected

Equivale a checked en los select, sirve para establecer una opción por defecto

autofocus

Para poner el foco por defecto al cargar el formulario
